## Known Issues / Watch Points

- GitHub Pages has a 1–2 min cache delay after each push — hard refresh with `Ctrl+Shift+R` if changes don't appear
- The `presentations` repo is currently public — any file pushed to `main` will be live
- Slide scaling is tuned for a large desktop screen; on smaller screens text may need further adjustment via the `width`/`height`/`font-size` values in the Reveal.js init block at the bottom of the HTML file
